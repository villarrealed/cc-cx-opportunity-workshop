# CC/CX Opportunity Identification Workshop

A 1-hour module within a half-day workshop, designed for **generalist Sales Engineers** with little to no prior contact center or customer experience expertise.

The workshop walks an SE through five AI-assisted prompts inside a single chat thread, producing four ready-to-use account artifacts:

1. **Customer + AI pre-brief** — top 2 opportunities ranked, with quantified financial impact
2. **Chosen opportunity + use case** — with quantified outcome and Cisco solution mapping
3. **Persona-tailored discovery questions + introductory email** — anchored to the chosen opportunity
4. **Customer-facing proposal** — in the format that fits the customer's buying motion

Plus static teaching content for opportunity identification, qualification, specialist routing, and meeting scaffolding.
